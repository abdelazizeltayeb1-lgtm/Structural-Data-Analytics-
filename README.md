# Structural Health Monitoring & Data Analytics Tool

This project was developed during my **Programming for Engineers Diploma** to address the challenge of analyzing unorganized engineering datasets, specifically in environments where physical laboratory access is limited.

### Key Technical Features:
*   **Automated Analytics:** Performed stress and displacement correlation analysis using **NumPy** and **Pandas**.
*   **Engineering Visualization:** Developed custom structural integrity monitoring profiles using **Matplotlib**.
*   **Data Structuring:** Transformed raw, unorganized sensor data into actionable insights for resilient infrastructure management.
